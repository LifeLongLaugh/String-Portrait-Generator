Transform photos into stunning string art with physically-accurate canvas rendering

Algorithmic Artistry

Numerical parity between brightness buffer processing and thread rendering with vanilla JavaScript

Quick Start

  1. Open the App to run the string-art generator
  2. Load your favorite photo
  3. Adjust the oval to match your board
  4. Position the image under the fixed oval
  5. Configure nails, thread thickness, and stroke count
  6. Generate your unique string portrait

Key Features

  • High-resolution backing buffer rendering
  • Export thread sequence (.txt) and final artwork (PNG)
  • Simple UI: nail count, max strokes, thread thickness
  • Drag-to-position image alignment
  • Physically-accurate canvas implementation
  • Black thread simulation for classic look

Implementation Notes

The application uses large internal backing buffers (high RES) to preserve physical accuracy. For optimal performance with large RES or high nail counts, consider these tips:

  • Line pixel precomputation planned for future versions to accelerate scoring
  • Accuracy is preserved by only altering the scoring path, not the commit path
  • Performance improvements focus on low-res scoring while maintaining high-res commit
